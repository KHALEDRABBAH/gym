# ⚡ Life Command Center — 90-Day Transformation PWA

A comprehensive Progressive Web App (PWA) designed to track and manage a complete 90-day life transformation journey. Built with vanilla HTML, CSS, and JavaScript — no frameworks needed.

## 🌟 Features

- **📊 Dashboard** — Overview of all life metrics, water tracker, streak badges, motivational quotes, monthly calendar view
- **✅ Daily Todo** — Daily checklist with categories (prayer, fitness, nutrition, health, study, habit) + journal
- **🔥 Habit Tracker** — Weekly & monthly heatmap-style habit tracking
- **📈 Progress Tracking** — Body measurements, weight, body fat %, muscle %, sleep, mood logging
- **📉 Analytics** — Charts for body composition, habit completion rates, daily scores, study/sleep trends
- **🕌 Prayer Tracker** — 5 daily prayers, mosque attendance, Quran reading log
- **📚 Study & Pomodoro** — Focus timer with presets (25/45/60 min), session tracking, study log
- **💪 Fitness** — Daily workout schedule (PPL split), workout log with weight/rep tracking, exercise tutorials with YouTube videos
- **🎬 Exercise Library** — Full exercise database with video tutorials, form cues, and common mistakes
- **🥗 Nutrition** — Meal plans, grocery list, supplement stack, nutrition rules
- **🎓 Learning Hub** — Curated courses from Coursera, edX, YouTube for CS & AI
- **📍 Transformation Roadmap** — Phase-by-phase body fat reduction guide with visual change descriptions
- **🎨 Lifestyle System** — Posture correction, cardio, mobility, daily routines, dopamine management, confidence building
- **📋 Weekly Review** — Wins, failures, lessons, improvements, and focus for the next week
- **🏆 Achievements & XP** — Gamification system with levels, XP points, and unlockable achievements
- **⚙️ Settings** — Profile settings, data export/import (JSON), auto-backup system

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom design system with CSS variables, dark/light themes, responsive design
- **Vanilla JavaScript** — No frameworks, all logic in a single file
- **Chart.js** — Data visualization
- **Service Worker** — Offline support (PWA)
- **localStorage** — Client-side data persistence with auto-backup

## 🌐 Language Support

- 🇬🇧 English
- 🇪🇬 Arabic (RTL support)

## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/life-command-center.git
   ```
2. Open `index.html` in your browser — that's it!

### Deploy to GitHub Pages

1. Go to your repo **Settings** → **Pages**
2. Set source to **main branch** → `/ (root)`
3. Your app will be live at `https://YOUR_USERNAME.github.io/life-command-center/`

## 📱 Install as PWA

1. Open the app in Chrome/Edge
2. Click the install icon in the address bar (or "Add to Home Screen" on mobile)
3. The app works offline after the first visit
